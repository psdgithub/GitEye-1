What is GitEye?

GitEye is a simple-to-use, graphical Git client that provides central visibility into essential developer tasks such as defect tracking, Agile planning, code reviews and build services.

GitEye is built on the Eclipse Rich Client Platform, the Eclipse EGit client and JGit library. While it uses the same framework and low level libraries as the Eclipse IDE, it is not intended to be an IDE itself. It leverages Mylyn and the CollabNet Desktop for ALM functionality.

Why use GitEye?

GitEye is free and lets you access any Git repository, cloud-based or on-site. It runs on most platforms (Windows, Mac and Linux) and provides you a unique Git client experience and range of features.
  • It is a full Git client with easy access to common Git actions such as clone, push, pull, merge, rebase, stash and more
  • Git History, Reflog and Staging views are available
  • It supports CollabNet CloudForge
  • It fully supports TeamForge ALM and Agile capabilities
  • It is GitHub-ready, with support for pull requests, issues and gists
  • It allows you to monitor Hudson and Jenkins builds
  • It supports Gerrit code reviews
  • You can optionally install Mylyn connectors such as Bugzilla, Trac, JIRA and Rally from within the application
The public project for GitEye is at http://www.collab.net/giteyeapp.
