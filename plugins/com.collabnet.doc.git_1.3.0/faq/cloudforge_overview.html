CloudForge overview

CloudForge enables you to provision and manage your development projects across languages, tools, and platforms, integrate and host Subversion and Git repositories under one roof, connect to popular Agile development tools, and deploy code to your private or public cloud.

Cloud development and deployment services

CloudForge provides hosted services that you can instantly provision for your development projects:
  • Source Code Management — Subversion (SVN), Git
  • Agile ALM — CollabNet TeamForge Project
CloudForge also provides integrations with other tools and a direct deployment framework to PaaS and private data center environments. See Why CloudForge? for more information.

CloudForge plans and pricing

The plans start at Free which includes unlimited projects, users and repositories.
Note: CloudForge projects are private.
For complete plan details, see cloudforge.com/pricing.

Using CloudForge with GitEye

Consider an example scenario where you have a small development team and want to try out CloudForge:
  • You (or any one person in your team) must first sign up for a CloudForge account and create the organization.
  • Set up your project and services.
    Tip: Available for free: unlimited TeamForge projects (via CloudForge and within GitEye), TeamForge user accounts and repositories, and 2 GB of storage.
  • Add the user accounts for the rest of your team so that you can all work together on the same project.
  • Have each team member connect their GitEye to CloudForge using the credentials they are given. (This will take care of the registration requirement for GitEye.)
For questions or comments, contact Support or check out the knowledge base: help.cloudforge.com.
