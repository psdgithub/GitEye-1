Export data

To use the information contained in tracker artifacts and tasks in other applications, export selected query results or planning folder data.

  1. To export data from a planning folder, open it in the Planning Board and click the Export Data icon.
    1. In the Export Data dialog, select the export format, the location to save the data, and the fields you want to export.
    2. Click OK.
  2. To export the results of a specific tracker or task query, select it in the CollabNet Sites tree.
    1. From the context menu, select Export Data.
    2. In the Export Data dialog, select the export format, the location to save the data, and the fields you want to export.
    3. Click OK.
Note: You can also export data for specific artifacts by selecting one or more artifacts from a planning folder or from a query.
