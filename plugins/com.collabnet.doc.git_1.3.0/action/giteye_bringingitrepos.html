Bring your Git repositories into GitEye

To start working with your Git repositories in GitEye, you can add an existing local repository, clone a remote repository or add and initialize a new repository.

  • Add a local repository to GitEye
    If you've already been working with Git using the command line or some other client, you can easily add your repositories to GitEye.
  • Clone a remote Git repository
    To clone a Git repository from CloudForge, GitHub or some other Git server, initiate the clone wizard and provide the location of the remote repository, the branches to clone, and the local destination for the repository.
  • Create a new Git repository
    You would create and initialize a new Git repository when you start a new project.
