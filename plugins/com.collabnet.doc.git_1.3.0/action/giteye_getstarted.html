Get started with GitEye

Get started by adding your Git repositories and TeamForge and CloudForge sites to GitEye.

  • Get GitEye
    GitEye is available as a free download. After the 30-day trial period, you need to register GitEye to continue using it.
  • GitEye Dashboard
    The Dashboard is the place where you add and clone Git repositories, identify yourself for Git commits, configure settings that control client behavior for Git and SSH, and check for product upgrades.
  • Generate an SSH key
    It is common to access Git repositories using SSH keys.
  • Bring your Git repositories into GitEye
    To start working with your Git repositories in GitEye, you can add an existing local repository, clone a remote repository or add and initialize a new repository.
  • Perform Git actions
    When you add or clone a Git repository in the Dashboard, it becomes available in the Git Repositories view which provides easy access to common Git functions.
  • Plan and track projects
    Add a TeamForge site in the CollabNet Sites view to make use of all the ALM and Agile functionality that TeamForge provides.
  • Monitor builds
    You can monitor builds from a Hudson or Jenkins continuous integration server by adding the build server in the Builds view. You will receive notifications when a build runs. You can examine the changes, test results and log files.
  • Review code
    You can visualize the details of Gerrit code reviews, monitor changes and act upon them within GitEye. You can pull and push changes to and from Gerrit and update existing reviews as well.
