Update document data

When updating a document, you can upload a new file, or change the document's name, description, or lock status.

  1. In the CollabNet Sites view, right-click a TeamForge document and select Update Document.
  2. In the Update Document window, make your changes.
    1. Update the name or description by entering new text in the appropriate field.
    2. Change the status value if appropriate.
    3. To prevent others from editing the document, select Lock Document.
    4. Select a file to upload, by browsing to it or entering its URL.
      Note: When updating, you can only select a single file at a time.
    5. Enter a version comment to describe the changes made, if appropriate.
    6. Click OK.
The document is now updated.
