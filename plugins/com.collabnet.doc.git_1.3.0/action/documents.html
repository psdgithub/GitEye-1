Document your work

The document management system is a centralized repository for creating, storing, and managing information about a project.

  • What is a document?
    For CollabNet TeamForge purposes, a document is any file that you track with a document artifact.
  • Create a document folder
    Document folders, like file directories, allow you to organize your documents in a logical, hierarchical manner using labels that are meaningful to you and your organization.
  • Create a document
    To create a document, you upload a file in the root folder or any other document folder.
  • Edit a document
    While editing a document, you can update its data and associate it with other TeamForge items.
