Manage tasks with the Agile task board

GitEye supports an Agile task board for TeamForge.

Task boards are an important tool in the Agile process. They help the team to focus on the work at hand in the current sprint and feed progress data back into the system, which helps produce the burndown and burnup charts.
Note: The task board is not a sprint-planning tool; it is more of a sprint-progress tool.
In GitEye, use the planning board for planning the stories that will be tackled in the sprint. During the sprint, team members can use the task board to break down the stories into tasks and then progress the tasks, and the story, towards completion.
  • Configure the task board
    TeamForge project administrators can configure the task board. Once that's done, projects members can use the task board.
  • Use the task board
    During a sprint, TeamForge project members can create tasks for new stories, assign and highlight tasks, update effort, and drag and drop tasks across swimlanes as they progress.
