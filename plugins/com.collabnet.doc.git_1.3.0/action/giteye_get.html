Get GitEye

GitEye is available as a free download. After the 30-day trial period, you need to register GitEye to continue using it.

Pre-requisites
  • You must have Java 1.6 (or later) installed. We recommend the JRE from Sun/Oracle although OpenJDK 7 also works well.
  • We recommend having 1 GB of available RAM.

Install GitEye

  1. Get the free download from www.giteyeapp.com. GitEye is delivered as a zip file. 32 and 64-bit versions are available for Windows, Mac and Linux operating systems. If you are on a 64-bit OS but are using a 32-bit JRE, get the 32-bit version of GitEye.
  2. Create an empty folder where you want to install GitEye and extract the zip file into that folder.
  3. Run GitEye.
    The launcher will typically find Java automatically, but if it cannot, or you need to control the specific JRE that it uses, you can edit the giteye.ini file. For example, if you have the 32-bit JRE in your Windows system PATH, and want to use the 64-bit version of GitEye, edit giteye.ini to include the following: 
    -vm 
C:/Program Files/Java/jre7/bin/javaw.exe
    Tip: For OSX 10.8 (Mountain Lion): If you have the Gatekeeper feature enabled, which it is by default, you will not be able to simply run GitEye after installation. To run GitEye, right-click on the launcher and click Open. This will present a dialog where you can confirm that you want to run the application even though it has not been digitally signed with an Apple Developer certificate. You only need to do this the first time you run the application.

Register GitEye

  1. After the trial period of 30 days, you must take one of the following actions to be able to keep using GitEye:
    • If you are an existing TeamForge or CloudForge user, just connect to that server
    • Sign up for a free CloudForge trial
After you've installed GitEye, you may want to customize the window layout. Here's a video that shows how you can do this: http://www.youtube.com/watch?v=eq40iJo1OpA
