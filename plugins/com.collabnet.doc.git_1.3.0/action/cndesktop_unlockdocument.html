Unlock a document

To work with a document that someone else has locked, you must unlock the document.

Only a document administrator and the document creator can unlock a document.
  1. In the CollabNet Sites view, right-click a TeamForge document and select Update Document.
  2. In the Update Document window, confirm that the lock status is currently Locked.
  3. Clear Lock Document.
  4. Click OK.
The document is now unlocked. All project members with the Document Edit permission can now update the document.
