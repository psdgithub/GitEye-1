Create a new Git repository

You would create and initialize a new Git repository when you start a new project.

  1. In GitEye's Dashboard, click Create Repository.
  2. In the Create a Git Repository dialog, specify the parent directory and name for your Git repository.
  3. If you only want to push changes to this repository from a different one, select the Create as bare repository option.
  4. Click Finish.
The repository is added to the Git Repositories view where you can start working with it.
