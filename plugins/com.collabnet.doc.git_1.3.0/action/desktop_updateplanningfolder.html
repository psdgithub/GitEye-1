Update a planning folder

A planning folder is a dynamic representation of a changing situation. From time to time, you'll want to update its name, description and scheduling parameters to reflect changes in the underlying work.

  1. In the CollabNet Sites view, right-click a planning folder and select Update Planning Folder.
    Tip: If you're working with the Planning Board, double-click the planning folder to update its attributes.
  2. Give the planning folder a new name and, if needed, a new description.
  3. Revise the start or end date of the planning folder to match any changes in your project's real schedule.
  4. Click OK.
