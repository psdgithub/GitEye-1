Update release attributes

As your release progresses, you should update its maturity level and status. You can also update its name or description if required.

  1. In the CollabNet Sites view, right-click a TeamForge document and select Edit Release.
  2. In the Edit Release window, make your changes.
    1. Update the name or description as required.
    2. Identify the maturity level of the release by selecting a predefined value such as "Early Access" or "Pre-general Availability".
    3. Choose a new status for the release. A release can be in Active or Pending status.
    4. Click OK.
The release attributes are now updated.
