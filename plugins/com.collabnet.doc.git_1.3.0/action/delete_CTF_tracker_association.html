Delete a tracker association

When a tracker artifact is no longer related to some other CollabNet TeamForge items, you can remove the association.

  1. If you're working in the artifact editor, expand the Associations section, and select a tracker association from the table.
  2. Click Delete. In that table row, a "-" icon appears adjacent to the association details.
    Note: The deletion is stored locally until you submit your changes to the server. If you have deleted the association in error, you can click Undelete to remove the deletion.
  3. Click Submit. The "-" icon that was displayed for that association, disappears. If you do not submit your changes, the next time you open Eclipse, the association is displayed with the "-" icon.
Tip: You can also delete an association in the CollabNet Sites view by expanding an artifact's Associations node and choosing the Delete association context menu option.
