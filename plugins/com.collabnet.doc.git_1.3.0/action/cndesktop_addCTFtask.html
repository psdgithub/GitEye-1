Create a task

Create a task when you identify a unit of work that you want to track to completion as part of your project.

  1. In the CollabNet Sites view, right-click a TeamForge task folder and select New > Task .
  2. On the New Task page, provide a name for the task.
  3. Assign the task a priority between 1 (highest) and 5 (lowest).
  4. Assign the task a status.
    • Not Started
    • OK
    • Warning
    • Alert
    • Complete
  5. Give the task a start and end date by making a selection from the calender.
  6. In the Estimated Hours field, write the time you think is needed to complete the task.
  7. Enter a description for the task.
  8. Assign the task to an individual, by selecting a project member in the Assigned To field.
  9. Click Submit.
