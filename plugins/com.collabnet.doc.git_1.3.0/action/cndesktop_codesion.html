Work with CloudForge

CloudForge is an easy-to-use development Platform as a Service (dPaaS) delivered by CollabNet Cloud Services.

  • CloudForge overview
    CloudForge enables you to provision and manage your development projects across languages, tools, and platforms, integrate and host Subversion and Git repositories under one roof, connect to popular Agile development tools, and deploy code to your private or public cloud.
  • Add a CloudForge site
    Whether you're a developer or project manager, you can enhance your CloudForge experience by adding your site to GitEye. Once you add your site, you can add new projects and services or continue with your existing setup while getting all the additional benefits GitEye provides.
  • Add a CloudForge project
    A CloudForge project is a container where you can add services such as TeamForge, Subversion and Git, and project members who can access these services.
  • Add CloudForge users
    Before your team can start work on a project, you need to add them as project users.
