Import a query

GitEye stores exported query definitions in XML files with the .tfq extension. Depending on the location of these files, you can import query definitions from the file system or from TeamForge Documents.

  1. In the CollabNet Sites view, right-click Site-wide queries and select Import query definition.
  2. Depending on where you want to import your query from, do one of the following:
    • Select File, click Next, and browse to the required query definition file in your file system.
    • Select Document, click Next, and follow the wizard to select a query definition document.
  3. Click Finish.
