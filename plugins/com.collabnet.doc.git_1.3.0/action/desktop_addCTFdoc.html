Create a document

To create a document, you upload a file in the root folder or any other document folder.

  1. Start with either of these steps:
    • In the CollabNet Sites view, right-click a TeamForge document folder and select New > Document .
    • Drag a system file and drop it onto a TeamForge document folder in the CollabNet Sites view.
    The Create Document window appears. If you used drag-and-drop, the name of the document is populated.
  2. By default, the document is given the name of the file you are uploading. To give the document a different name, de-select Use file name and provide a new name.
  3. Provide a description for the document.
  4. Identify the status of the document.
    • Draft — The document is not yet completed.
    • Final — The document is finished.
  5. If you want to prevent others from editing the document, select Lock Document.
  6. Select the document creation method:
    • For the Upload from file or directory field, browse to the desired file. If you used drag-and-drop, the name of the document is already populated.
    • Specify the URL for the document.
  7. Write any comments you may have about this version of the document.
  8. If you want to associate the document with another TeamForge item, enter the ID of that item. For example, artf1010.
  9. Enter a description explaining the association.
  10. Click OK to save the document.
