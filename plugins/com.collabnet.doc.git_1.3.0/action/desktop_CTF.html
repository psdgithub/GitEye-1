Work with TeamForge

TeamForge is CollabNet's Agile ALM platform for distributed development. With TeamForge, you get software configuration management (SCM), continuous build and test, issue tracking, project management, lab management, and collaboration tools, all in a single tightly-integrated, cloud-based platform that empowers distributed teams to deliver great software.

  • Add a new site
    To start working with your TeamForge projects within
  • Customize the CollabNet Sites view
    GitEye favorites are TeamForge projects and tools that you use frequently. You can customize your CollabNet Sites view by selecting the projects and tools you want to see while browsing the site.
  • Track your work
    Use the issue tracking system to track and manage everything from requirements, user stories, and feature requests to defects, build issues, and customer support cases.
  • Plan and monitor your project
    The dynamic planning features of TeamForge help you effectively utilize Agile processes in your project.
  • Release products
    Publish the ouput of your project to selected audiences as packages and releases.
  • Work on tasks
    Manage your project plan by defining units of work and assigning them to project members.
  • Document your work
    The document management system is a centralized repository for creating, storing, and managing information about a project.
  • Work with Gerrit
    When your site has the TeamForge Git integration installed, you will see a Gerrit node under your TeamForge project node in the CollabNet Sites view.
  • See an item's association tree
    To see a cascading chain of all items associated with a single, selected TeamForge item, bring up its Association Tree.
  • Export data
    To use the information contained in tracker artifacts and tasks in other applications, export selected query results or planning folder data.
