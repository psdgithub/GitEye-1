GitEye Dashboard

The Dashboard is the place where you add and clone Git repositories, identify yourself for Git commits, configure settings that control client behavior for Git and SSH, and check for product upgrades.

Git Actions tab

SSH Settings

This option lets you create SSH key pairs, manage them or point to existing keys. For more information, see Generate an SSH key.

Add Repository

With this option, you can add a local Git repository and start working with it in GitEye. For more information, see Add a local Git repository.

Clone Repository

This option initiates the wizard for cloning a Git repository from TeamForge, CloudForge, GitHub or any other Git repository on a remote server. For more information, see Clone a Git repository.

Create Repository

Use this option to create a new local Git repository. For more information, see Create a new a Git repository.

Git Settings

Use this option to specify how you want to be identified in Git commits, as well as to configure settings for Git history, project resources and more.

Settings

Use this option to automatically check for product updates and install them.

CloudForge tab
